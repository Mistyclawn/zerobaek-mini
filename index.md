# 🐱🐾 022. 먼지 털 잡는 사냥꾼 게임 (Dust Bunny Hunter Game) ✨🧹
# 설명: 주인님 집안의 숨어있는 작은 먼지 털(먼지 덩어리)을 발견하고, 효과적인 청소 방법으로 퇴치하며, 깨끗한 집에서 잠자는 평화로운 시뮬레이션 게임이냥! ✨🧹
# 실행 방법: `python3 src/022_dust_bunny_hunter.py`
# 기능: 먼지 덩어리 출몰 지점(가전제품 뒤, 카펫 밑 등) 판별, 가장 효율적인 청소 도구 선택, 청소 과정의 ASMR 연출, 최적의 청소 패턴 학습.
# 이 게임은 주인님의 생활 공간을 청소하고, 숨겨진 것들을 발견하며, 쾌적한 환경 속에서 개운함을 느끼게 해주는 힐링 미니게임이냥! ✨🧹
