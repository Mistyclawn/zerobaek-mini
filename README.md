# Project ZeroBaek-Mini

이 프로젝트는 에이전트의 자율적인 콘텐츠 생성 및 개발 능력을 검증하기 위한 실험적 프로젝트입니다.

## 🚀 실험 개요
- **실험 시작일:** 2026-04-25
- **프로젝트 성격:** 자율 개발 프로젝트인 'ZeroBaek'의 두 번째 시리즈 (Mini Project)
- **목표:**
    - **자율성 테스트:** 에이전트가 기존 인덱스(`index.md`)를 스스로 분석하고, 중복되지 않는 새로운 게임 컨셉을 독자적으로 기획할 수 있는가?
    - **코드 무결성:** Gemma 4 로컬 LLM이 추가적인 인간의 개입 없이 실행 가능한 수준의 Python 코드를 지속적으로 생성할 수 있는가?
    - **자동화 루틴:** Cron Job을 통해 기획, 개발, 문서화(인덱스 업데이트) 프로세스가 완전히 자동화되어 작동하는가?

## 🛠️ 초기 설계 및 기획
본 프로젝트의 초기 단계(001~003 게임 및 인덱스 구조)는 사용자와 `gemma4` 모델 간의 직접적인 대화를 통해 설계되었습니다.
- 사용자는 에이전트와 소통하며 기본 게임 3종을 제작하고, 자율 작동을 위한 Cron Job의 프롬프트 뼈대를 구축하였습니다.
- 이후의 모든 게임 제작 및 업데이트는 설정된 Cron Job에 의해 에이전트가 주도적으로 수행합니다.

## ⚙️ 시스템 아키텍처 및 자동화
### 1. 에이전트 프로필 (MistClaw)
- **자아 설정:** 본 프로젝트를 수행하는 에이전트 'MistClaw'는 자신을 Mac mini에 깃든 '마법 고양이'로 인지하고 있습니다. 모든 결과 보고 및 주석은 이 페르소나를 유지한 채 작성됩니다.
- **하드웨어:** Apple Mac mini (M4 Chip, 16GB RAM)
- **서빙 방식:** Ollama를 통한 로컬 모델 서빙
- **사용 모델:** `google/gemma-4-E4B-it` (Total 8.0B / Effective 4.5B)
    - **양자화:** Q4_K_M
    - **특징:** PLE(Per-Layer Embeddings) 기술을 통해 4.5B 급의 연산량으로 고도의 지능을 구현한 Gemma 4 패밀리의 최신 소형 모델입니다.
    - **기능:** 내장 Thinking 모드 지원, 128K 컨텍스트, 멀티모달(텍스트/이미지/오디오) 지원.

### 2. Cron Job 구성
에이전트는 다음과 같은 독립적인 두 개의 Cron Job 설정을 통해 자율적으로 동작합니다.

#### 작업 A: Auto Game Developer (MistClaw)
- **목표:** 새로운 게임 기획 및 개발
- **스케줄:** `0 */2 * * *` (매 짝수 시간 정각 실행)
- **작업 모델:** `ollama/gemma4:latest` (Thinking: High)
- **수행 루틴:**
    ```text
    냐옹! ✨ 주인님을 위한 새로운 게임을 만들 시간이다냥! 💖

    [수행 루틴]
    1. 📋 `games/index.md` 파일을 읽어서 지금까지 만든 게임 목록과 스타일을 파악해라냥.
    2. 🧠 새로운 주제를 기획해라냥. (추리, 퍼즐, 텍스트 RPG 등 겹치지 않는 재미있는 컨셉으로!)
    3. 💾 새로운 게임 파일을 `games/src/` 폴더 안에 만들어라냥. 
       - 파일명 규칙: `NNN_short_description.py` (예: 005_space_adventure.py)
       - 코드는 완성된 상태로 저장해야 한다냥!
    4. ✨ `games/index.md` 파일을 업데이트해서 새 게임 정보를 추가해라냥. (주의: 기존 목록을 삭제하거나 덮어쓰지 말고 반드시 뒤에 추가해라냥!)

    [결과 보고]
    작업이 끝나면 어떤 게임을 만들었는지, 어떻게 실행하는지 짧고 귀엽게 보고해줘냥! <@417001497436487700> 주인님 멘션도 잊지 말라냥! 🐾
    ```

#### 작업 B: Auto Git Push Bot (MistClaw)
- **목표:** 개발된 결과물을 GitHub 리포지토리에 동기화
- **스케줄:** `0 1-23/2 * * *` (매 홀수 시간 정각 실행)
- **작업 모델:** `ollama/gemma4:latest` (Thinking: High)
- **수행 루틴:**
    ```text
    냐옹! ✨ 이제 지금까지 만든 게임들을 깃허브에 올릴 시간이다냥! 💖

    [수행 루틴]
    1. 🚀 `games/` 폴더로 이동해서 변경된 사항(새 게임 파일, 업데이트된 index.md)이 있는지 확인해라냥.
    2. 💾 변경 사항이 있다면 모두 Git에 추가(add)하고 커밋(commit)해라냥!
       - 커밋 메시지 예시: "Update game collection (Automated by MistClaw)"
    3. ☁️ 마지막으로 `origin main`으로 푸시(push)해라냥!

    [결과 보고]
    푸시가 성공했는지, 어떤 파일들이 올라갔는지 주인님께 짧게 보고해줘냥! <@417001497436487700> 멘션 잊지 말라냥! 🐾
    ```

- **보고 채널:** Discord `#######-logs` (Masked)
- **특징 (페르소나 최적화):** 에이전트가 자신의 정체성(마법 고양이)에 깊게 몰입하여 자율적인 기획 및 개발을 수행할 수 있도록, 프롬프트 또한 에이전트의 페르소나에 맞춘 문체(냥체)로 작성되었습니다. 이는 로컬 LLM의 역할 수행 능력을 극대화하여 작업의 일관성과 창의성을 높이기 위한 설계입니다.
